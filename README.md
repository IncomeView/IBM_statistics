Este repositório documenta minha jornada no curso **Statistics for Data Science with Python**, oferecido pela IBM.  
Aqui organizo anotações, notebooks autorais, experimentos e o **projeto final oficial** do curso.

O objetivo é consolidar raciocínio estatístico, registrar aprendizados e construir um portfólio técnico sólido.

<br>

O projeto oficial do curso está disponível em:

~~~

~~~

- Este projeto aplica:

    - Estatística descritiva  
    - Visualização
